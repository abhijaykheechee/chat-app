## **Run Locally**

### Set environment variables

```js
PORT=[YOUR_PORT_CHOICE]
MONGO_DB_URI=[YOUR_MONGO_DB_URI]
JWT_SECRET=[YOUR_JWT_SECRET]
NODE_ENV=...
```

### Build app

```shell
npm run build
```

### Start app

```shell
npm start
```
